# Vendor Assets

This repository intentionally does not redistribute the raw `JC3636K718_knob_EN/` vendor bundle or similar third-party board packages.

Keep the public repository limited to:

- original project source code
- project-written documentation
- screenshots and assets created specifically for this repository
- derived hardware notes such as [`JC3636K718.md`](./JC3636K718.md)

Do not publish vendor-origin materials here unless you have separately verified that redistribution is allowed.

Typical examples that should stay out of the public tree unless licensing is clear:

- vendor IDE projects and demo apps
- prebuilt binaries, flashing tools, and recovery images
- PDFs, schematics, and manuals copied from the vendor bundle
- logs or configuration dumps taken from a personal machine
- extracted app artwork, interface assets, or brand logos

For this controller project specifically:

- do not bundle official La Marzocco logo files in the repository
- do not ship an official La Marzocco logo as the default UI header
- do not use an official vendor logo in README hero images, repo previews, or promo screenshots
- describe compatibility in text instead
- treat any uploaded controller logo as user-provided local data, not as a repo asset
